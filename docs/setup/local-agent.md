**QApilot** allows you to run tests on your local machines. To run the tests locally, you need a small QApilot Local Agent (Java) utility running on the machine for test orchestration, i.e., queueing tests, running the tests, fetching the test results, etc. For more information, Refer to [Local Recording/Execution](/detailed-documentation/setup/local-recording-execution)

***

## Prerequisites

* You should have a QApilot Local agent on your machine.
* Enable Developer options on your mobile device.

***

## Updating QAPilot Local Agent of the New Version

1. Navigate to **Local Agent** and click **Download Agent**.

![](https://docs.qapilot.io/files/YZkNOxYmsh9oxuTdgZN2)

2. It will redirect to the below window. Check the new and old versions of the local agent executor files for Windows, Max, and Linux devices.

![](https://docs.qapilot.io/files/2bh30AGDmbbu1mIwLAH4)

3. Also, you can refer to the document and video demo for installation instructions on the devices listed in the top right corner.
