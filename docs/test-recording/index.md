You can quickly record test steps by capturing user interactions, such as clicking buttons, entering text, navigating through screens, etc. They help users create elements that interact with the application during testing.

***

## Prerequisites

1. Ensure you create a [Project ](/detailed-documentation/projects)before recording the test steps.
2. Establish a [Local/Cloud Device](/detailed-documentation/setup/system-requirements) connection setup.
3. You should know how to create a [Test Case](/detailed-documentation/qapilot-terminology#test-case)

***
