Project settings typically refer to the configuration options or parameters available for testing a project, allowing you to control aspects of the project environment, tools, automation workflows, and execution. These settings help tailor the project to meet specific requirements, manage integrations, set up environments, and configure workflows. The Project Settings has one tab.

[**Integrations**](/detailed-documentation/project-settings/integrations)

Integrations will be helpful to connect various virtual devices to a specific app as a part of the Project. This will facilitate the app's feasibility for a wide range of mobile devices concerning the functionality of the app. The integration setup will simplify how well a newly developed application is compatible with a specific mobile device. This testing is done on many mobile devices in a virtual environment as a part of Cloud infrastructure.
