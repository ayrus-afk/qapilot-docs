Account Settings

How To Manage a QA Pilot Account?​

  1. The bottom-left corner of the Dashboard page will display the Username.

  1. Click on the Account Settings below the Username.

  1. The above menu shows the Account Settings and Logout options along with the Username and Mail ID.
  2. Select the Account Settings option.

Account Settings deals with updating User Profile information. It displays Username, email ID, and Mobile number details.

  1. The below screen with User Profile information is shown below.

Manage Account deals with updates to the User Profile information. It displays Username, Email id, Last Login details, and Mobile number details.

This section facilitates the upload of a required Profile picture of the user.

The new Password update is also changed in this section as shown above. For security reasons, if the User/Admin wishes to create a new password the Change Password feature will be helpful.

Updating the Profile Picture and Other details​

  1. On the Manage Account page, click the Upload icon to update the profile picture against the Username as shown below.

  1. The Admin/User is redirected to the Open dialog box to pick the picture.
  2. Click Open. The new Profile Picture has been uploaded as shown below.
  3. An Edit icon is also shown beside the newly uploaded picture. The user can change the Profile picture as per the requirement.
  4. The updated picture is shown on the Dashboard page also.

Update the Profile Information?​

  1. On the Manage Account page, the User Profile basic information is updateable.

  1. In the Profile Information section, the Username and Mobile Number are editable.
  2. As per the requirement, modify these fields.
  3. Click the Update button to make the changes effective.

  1. The display of the above message confirms the update of details.

Change the Password​

  1. The user can provide a New Password in the respective field under the Password Information section as shown below.

  1. The system will prompt you to re-enter the password to confirm it.
  2. Click the Update button**.** The change of password is confirmed by the below message.

  1. Click the Logout button to exit from the interface.
